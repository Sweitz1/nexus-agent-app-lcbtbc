# Go Libraries & Modules

Top open source Go modules and frameworks.

## Web Frameworks
| Module | Description | Install |
|--------|-------------|---------|
| [Gin](https://github.com/gin-gonic/gin) | Fast HTTP framework | `go get github.com/gin-gonic/gin` |
| [Echo](https://github.com/labstack/echo) | High-performance framework | `go get github.com/labstack/echo/v4` |
| [Fiber](https://github.com/gofiber/fiber) | Express-inspired framework | `go get github.com/gofiber/fiber/v2` |
| [Chi](https://github.com/go-chi/chi) | Lightweight router | `go get github.com/go-chi/chi/v5` |
