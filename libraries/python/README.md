# Python Libraries

Top open source Python libraries across every domain.

## Web Frameworks
| Library | Description | Install |
|---------|-------------|---------|
| [Django](https://github.com/django/django) | Full-stack web framework | `pip install django` |
| [FastAPI](https://github.com/tiangolo/fastapi) | High-performance async API framework | `pip install fastapi uvicorn` |
| [Flask](https://github.com/pallets/flask) | Lightweight web framework | `pip install flask` |
| [Starlette](https://github.com/encode/starlette) | Async framework (FastAPI's core) | `pip install starlette` |
