# JavaScript / TypeScript / Node.js Libraries

## Frontend Frameworks & UI
| Library | Description | Install |
|---------|-------------|---------|
| [React](https://github.com/facebook/react) | UI component library | `npm i react react-dom` |
| [Vue](https://github.com/vuejs/vue) | Progressive UI framework | `npm i vue` |
| [Svelte](https://github.com/sveltejs/svelte) | Compiled UI framework | `npm i svelte` |
| [Next.js](https://github.com/vercel/next.js) | React meta-framework | `npx create-next-app` |
| [Nuxt](https://github.com/nuxt/nuxt) | Vue meta-framework | `npx nuxi init` |
| [Astro](https://github.com/withastro/astro) | Content-first framework | `npm create astro` |
